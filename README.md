# Поетичний складоріз

Браузерний інструмент для чернеток українських віршів: вводите текст і отримуєте таблицю складів із підсвіткою слова/складу та прив'язкою до позиції курсора.

## Можливості
- Розбиття кожного рядка на склади (наївно за голосними) з групуванням складів одного слова єдиним кольором.
- Слова з однієї приголосної «прилипають» до наступного складу (наприклад, «в лісі» → «_в лі_|сі»).
- Синхронний скрол textarea ↔ таблиця; активний склад автоскролиться у видиму область.
- Підсвічення складу за позицією курсора, підсвічення рядка таблиці та жирний лічильник складів у ньому.
- Ігнорування дефісів/тире всередині слова при підрахунку складів.

## Використання
- Відкрийте `index.html` у браузері локально — нічого встановлювати не потрібно.
- Ліворуч/угорі вводьте текст; праворуч/унизу одразу бачите таблицю складів.

## Обмеження
- Алгоритм складів наївний (за голосними); можливі похибки у складних випадках та винятках.
- Наголоси й рими поки не підтримуються.
- Дефіси вилучаються перед розбиттям; для специфічних правил переносу можуть бути розбіжності.

## Розробка
- Код повністю у `index.html` (HTML/CSS/JS без збірки). Просто редагуйте файл і оновлюйте сторінку в браузері.
